Проекты

Посмотрите некоторые из моих предыдущих проектов:

  • 1. Система управления задачами (Task Manager) Веб-приложение для управления задачами, позволяющее пользователям создавать, редактировать и удалять задачи.   • 3. Интернет-магазин (E-commerce Store) Полнофункциональный интернет-магазин, предлагающий пользователям возможность просматривать товары, добавлять их в корзину и оформлять заказы. В проекте реализованы функции аутентификации пользователей и администрирования товаров. Используется React для интерфейса пользователя, Node.js с Express для серверной части и PostgreSQL для управления базой данных.
